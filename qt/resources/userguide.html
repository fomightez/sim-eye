User Guide

Introduction

Aim

Gore Sim Eye is designed to take a fundus image, and produce a net which when printed, may be used to construct a simulation eye (or sim eye) suitable for teaching and learning in ophthalmology. The net is constructed of several petal-shaped sections known as gores, which may be conveniently joined to make an approximation of a spherical eye.

Use

The user selects a fundus image file. This image should be square (or very nearly square) and centred, and the fundus should be tight against the edge of the image. Common image formats such as JPEG and PNG are supported. The user must enter the angular extent of the fundus image. There are various options to control the gore net that is produced, e.g. the number of gores to use. The resulting gores may then be calculated and displayed, and finally saved as an image (again, in one of several common formats). This image may then be printed, cut out and assembled to form a sim eye.

Approach

The process involves two stages:

  1. the positions of pixels in the fundus image are transformed into corresponding positions in the eye, using a simplified optical model, to produce a "map" of the eye based on the input image;
  2. this map is projected into a series of gores using a standard map-making projection, and the gores are stitched together into a final image.

Using this tool

Open an image

From the File menu, select Open input image, or click the Open icon on the toolbar. This will open the file dialog: simply select your chosen fundus image and click OK. The image should be square (or very nearly square) and centred, and the fundus should be tight against the edge of the image. Common image formats such as JPEG and PNG are supported.

Settings

Adjust the settings using the controls in the main window. These are as follows (note that hovering over any control/icon will display a help "tooltip").

  • Image extent

    Angular size of the input image, in degrees, measured from edge to edge (i.e. a fundus image that covered the half-sphere would have an extent of 180 degrees).

  • Number of gores

    Number of gores to produce: a higher number will approximate a sphere better, but may be more difficult to assemble.

  • Retinal size

    Desired angular extent of the gores. Selecting 360 degrees will produce gores that make an entire sphere.

  • No-cut area

    Controls the angular size of the region in the centre of the gores where the gores are joined together (i.e. there are no cuts in this region). Selecting a bigger angle may make cutting more straightforward, but may also result in a slightly flattened region at the centre.

  • Rotation

    Degrees by which to rotate the fundus with respect to the petals. Rotation is specified as counter-clockwise: use negative degrees to rotate clockwise. This option is useful, for example, for ensuring that interesting parts of the image do not fall on or near the cut edges of a gore.

  • Quality

    Percentage of original image resolution: 100% corresponds to the maximum original image resolution. Using a lower quality setting will make calculation faster, but results in lower-resolution images. This option is useful for testing options.

Background colour

Set the background colour by selecting Choose background colour from the File menu, or clicking the background colour icon on the toolbar.

Gore!

Click the Gore button to process the image and produce the gores. The status bar will display the current stage of the calculation as it progresses. Once complete, the output is displayed in the preview window. To cancel the calculation, click the cancel button (you may have to wait a moment for the current stage to complete before the calculation is cancelled).

Save the image

Select Save as from the File menu (or click on the Save As icon on the toolbar) to choose a file name and location for saving the resulting gore image. To select the image format for the saved image, simply choose and enter the appropriate file extension, e.g. .jpg for JPEG, .png for PNG etc. The Save option will save the current output image using the current filename.

Close and Exit

Close the current image by selecting Close from the File menu, or by clicking the Close icon on the toolbar. Exit the application by selecting Exit from the File menu, or by clicking the Exit icon on the toolbar.
